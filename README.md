# Portafolio Personal

![Badge](https://img.shields.io/badge/Estado-Completado-brightgreen)

Este repositorio contiene mi portafolio personal, que incluye información sobre mis datos personales, intereses, proyectos, experiencia laboral, educación, contacto y redes sociales. El diseño está optimizado para ser **responsivo** y fue creado utilizando **HTML**, **CSS**, **SASS** y **JavaScript**.

---

## 🚀 Tecnologías utilizadas
- **HTML5:** Para estructurar el contenido de la página.
- **CSS3 y SASS:** Para estilizar la página y aplicar un diseño modular y fácil de mantener.
- **JavaScript:** Para agregar interactividad y funcionalidades dinámicas.
- **Media Queries:** Para asegurar que el diseño se vea bien en diferentes dispositivos (móviles, tabletas y escritorio).

---

## 🛠️ Instrucciones de instalación y uso
Este proyecto es una página web estática, por lo que no requiere instalación.
